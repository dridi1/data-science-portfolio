# Data Science Portfolio

A modern, responsive portfolio website for Dridi Sami - Data Scientist & ML Engineer.

## Features

- **Bilingual Support**: Seamless switching between English and French
- **Responsive Design**: Mobile-first approach with modern CSS
- **Interactive Sections**: Hero, About, Experience, Education, Projects, Contact
- **Modern UI/UX**: Clean design with smooth animations and transitions
- **Accessibility**: WCAG compliant with keyboard navigation support
- **Performance Optimized**: Fast loading with efficient asset management

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript (ES6+)
- JSON for content management
- Progressive Enhancement

## File Structure

```
/
├── index.html              # Main HTML file
├── styles/
│   ├── main.css           # Main stylesheet
│   └── responsive.css     # Responsive design
├── scripts/
│   ├── main.js           # Main application logic
│   └── language.js       # Language switching functionality
├── assets/
│   ├── images/           # Image assets
│   └── icons/            # Icon assets
├── data/
│   └── content.json      # Bilingual content data
└── README.md             # This file
```

## Getting Started

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Or serve through a local web server for full functionality

## Language Support

The portfolio supports two languages:
- English (EN) - Default
- French (FR)

Language preference is saved to localStorage and persists across sessions.

## Browser Compatibility

- Chrome 70+
- Firefox 65+
- Safari 12+
- Edge 79+

## Performance Features

- CSS Grid and Flexbox for layout
- Intersection Observer for animations
- Lazy loading for images
- Minimal JavaScript for optimal performance
- Responsive images

## Accessibility Features

- Semantic HTML structure
- ARIA labels and landmarks
- Keyboard navigation support
- Screen reader friendly
- High contrast support
- Reduced motion support

## Customization

Content can be easily updated by modifying the `data/content.json` file. The design can be customized through CSS custom properties in `styles/main.css`.
